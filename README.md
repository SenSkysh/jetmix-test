## Установка

Создать .env файл

Установить зависимости
```composer install```
Создать базу
```php artisan migrate```

## Тесты
```php artisan test```
После тестов база очищается

## Наполнение данными
```php artisan db:seed```

## Запуск
```php artisan serve```
